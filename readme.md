# Professor Ricardo Semerene — Educational Landing Page

Landing page project created to professionally present private math lessons.

## Sobre o projeto / About the project

Uma página estática moderna e responsiva, criada para destacar o trabalho do professor Ricardo Semerene. O design prioriza clareza, confiança e conversão, com chamadas visuais para contato via WhatsApp e apresentação de planos de estudo.

A modern responsive static page designed to highlight the work of Professor Ricardo Semerene. The layout emphasizes clarity, trust, and conversion, with visible WhatsApp contact buttons and service plans.

## Recursos principais / Key features

- Interface responsiva para desktop, tablet e mobile / Responsive layout for desktop, tablet, and mobile
- Menu mobile e navegação suave entre seções / Mobile menu and smooth scroll navigation
- Seção de planos e diferenciais do serviço / Plans and service differentiators section
- Integração com WhatsApp para contato rápido / WhatsApp integration for fast contact
- Animações simples de entrada ao scroll / Simple scroll-triggered animations
- Estrutura semanticamente correta para SEO / Semantic structure optimized for SEO

- Update the WhatsApp link in `index.html` with a real number in international format.
